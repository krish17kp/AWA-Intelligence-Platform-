# AWA Intelligence Platform

Evidence-first backend platform for collecting, preserving, and analyzing USDA Animal Welfare Act records.

## Current Focus

Phase 1 backend data foundation:

- APHIS-first source ingestion
- Raw source preservation
- SHA-256 hashing
- Provenance metadata
- Duplicate prevention
- Railway PostgreSQL for metadata
- Railway/S3-compatible bucket storage for raw source documents
- n8n orchestration for scheduled ingestion
